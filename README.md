# AI Smart Security Alarm System

## Overview
An AI-based smart security alarm system using Raspberry Pi 4 that detects motion and alerts the user.

## Components
- Raspberry Pi 4
- PIR Motion Sensor
- Buzzer
- LED
- Jumper Wires
- Breadboard

## Features
- Motion detection
- Alarm activation
- LED indication
- Future AI integration
